#  Probabilistic Machine Learning Summary

This repository contains a detailed implementation and summary of **Uncertainty Modeling** in Machine Learning, inspired by the lecture of [Eng. Mohamed Hammad](https://lnkd.in/gqyz4UTf).

##  Overview
Traditional ML models often provide a single point estimate without accounting for data noise. This notebook explores how to model **Heteroscedastic Uncertainty** to build more reliable and "honest" AI systems.

##  Key Concepts Covered
* **Homoscedastic vs. Heteroscedastic Noise:** Understanding why uncertainty isn't constant.
* **Probabilistic Output Layers:** Designing NNs that predict Mean ($\mu$) and Variance ($\sigma^2$).
* **NLL Loss:** Using Negative Log-Likelihood to capture data noise.
